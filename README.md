# S2S Protocol (TCP)

## Setup Instructions

1. Initialize Node.js project:
   ```bash
   npm init
   ```

2. Install dependencies:
   ```bash
   npm install @types/node -D
   ```

3. Configure package.json:
   - Add `"type": "module"` to your package.json

## Running the Project

1. Open two terminal windows

2. Start the server:
   ```bash
   node server.js
   ```

3. Start the client:
   ```bash
   node client.js
   ```
